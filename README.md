# forkingpaths-hydra

## How to use:
1. Start Forking Paths App
2. open hydra
3. make some new lines at the top
4. copy all of the code from the file forkingpaths-hydra.js
5. paste it at the top of hydra code
6. run the forking paths js code in hydra
7. comment out the FP js code in hydra
8. now you're free to run single lines or all without causing the fp code to restart
